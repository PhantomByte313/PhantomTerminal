# ◈ Phantom Terminal

A professional Python terminal emulator with a deep-black dark aesthetic.

## Run

```bash
pip install PyQt6
python3 main.py
```

## Shortcuts

| Shortcut | Action |
|---|---|
| `Ctrl+Shift+`` ` | New Terminal |
| `Ctrl+Shift+W` | Close Terminal |
| `Ctrl+Shift+C` | Copy |
| `Ctrl+Shift+V` | Paste |
| `Ctrl+Shift+F` | Find |
| `Ctrl+K` | Clear |
| `Ctrl+Shift+=` / `-` | Font size |
| `F11` | Fullscreen |

## Requirements
- Python 3.8+
- PyQt6
- Linux / macOS
